## Реализация учебного проекта "Банкомат"

Состоит из подпроектов:
#### rest
   
   Хранит и предоставляет данные о клиентах, счетах, картах.
   
#### web
   
   Реализует UI для работы с клиентскими данными.
   
#### common
   
   Содержит DTO для обмена между rest и web.
